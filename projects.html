Self Navigating Robotic Vehicle

For my junior design course, I was tasked with designing and building a self-navigating robotic vehicle. The project included embedded C programming on an Arduino, circuit design for color and object detection, and power management between multiple power sources.

Five Stage Pipelined Processor

I designed a five stage pipelined processor using VHDL. The processor runs on a small subset of the LEGv8 instruction set and is capable of structural and data hazard resolution with forwarding and stalling.

Analog Pulse Oximeter

For my analog circuits class, I designed a pulse oximeter that displays the user's heart rate on an oscilloscope. The device uses op-amps to create a bandpass filter, photodiodes to detect the user's pulse, and a voltage follower to give a clear signal to the oscilloscope.

Heads Up on the STM32L4 Discovery Kit

I designed a simple version of the mobile game Heads Up using the STM32L4 Discovery Kit. The game uses the LCD display a word on the board, and registers rotation using the onboard accelerometer. UART is used to send data to a monitor where the player can see their score, as well as other game information.
